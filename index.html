Crystal Nightmare

Crystal Nightmare is an online multiplayer tower defense game that I built using Unity3D with my friends Gabriel and Alana. You can download the game here: Download (Windows)

I wanted to extend support to OSX and Linux, but it turns out a lot of the Unity3D features that were used to build the game is obsolete. If I have time sometime, I'll try to fix all the issues and extend support.
